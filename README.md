# Lab 3 - Formulario tipo Login con React Hooks

## Descripción
Formulario de registro desarrollado en React que utiliza los hooks `useState`, `useEffect` y `useRef` para el manejo de estado, efectos secundarios y referencias del DOM.

## Campos del formulario
- Matrícula
- Nombre
- Apellidos
- Edad
- Universidad
- Carrera

## Hooks utilizados
- **useState** → guarda y actualiza los valores de cada input en tiempo real
- **useEffect** → detecta cambios en el formulario y ejecuta acciones al cargar la página
- **useRef** → pone el foco automáticamente en el campo de matrícula al iniciar

## ¿Cómo funciona?
Al llenar los campos y presionar el botón **Enviar**, se muestran los datos ingresados en una tarjeta debajo del formulario.
